# 🚀 AI Resume Analyzer

An intelligent web application that analyzes resumes, extracts key skills, calculates ATS score, and compares them with job descriptions to improve job readiness.

---

## ✨ Features

- 📄 Upload resume (PDF)
- 🧠 Automatic skill extraction
- 📊 ATS score calculation
- 🔍 Job description matching
- ✅ Matched skills identification
- ❌ Missing skills detection
- 📥 Download analysis report

---

## 🛠️ Tech Stack

- Frontend: Streamlit  
- Backend: Python  

---

## 📌 How It Works

- Upload your resume  
- Paste the job description  
- The app analyzes:
  - Skills in your resume.  
  - ATS score  
  - Match percentage  
- View matched and missing skills instantly  

---

## 🎯 Use Case

Helps students and job seekers:

- Improve resume quality  
- Identify missing skills  
- Prepare better for job applications
